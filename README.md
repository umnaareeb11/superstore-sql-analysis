 Superstore Sales Data Analysis — SQL Assignment

 Objective
Analyze retail sales data using SQL with filtering, aggregation, sorting, and business queries on the Superstore dataset.

 Dataset
- **Source:** Kaggle — Superstore Dataset
- **Rows:** 9,994 records | **Columns:** 21 | **Period:** 2014–2017

 Tools Used
- Python 3, SQLite (MySQL-compatible), pandas, Jupyter Notebook

 Key Insights
-  West region has highest sales — $725,457
-  Technology is most profitable — $145,454 profit
-  Standard Class is most used shipping — 5,968 orders
-  California is top revenue state — $457,687
-  Copiers are most profitable sub-category — $55,617
-  1,871 orders resulted in losses due to high discounts
-  Sales peak every Q4 (November–December)
